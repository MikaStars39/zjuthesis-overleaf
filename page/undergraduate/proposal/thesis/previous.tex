    \par {\zihao{4}\bfseries 二、指导教师对文献综述、开题报告、外文翻译的具体要求：}
}

\begin{enumerate}
    \item 文献综述：广泛查阅国内外关于人类数字偏差与大语言模型数值推理的相关文献，不少于20篇，其中外文文献不少于10篇，综述应系统梳理研究现状与不足。
    \item 开题报告：明确研究问题与目标，阐述研究方法与技术路线，制定合理的进度计划。
    \item 外文翻译：选译与课题密切相关的英文文献一篇，译文准确通顺，不少于5000字。
\end{enumerate}
